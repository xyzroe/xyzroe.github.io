OpenTherm - TTL.md
$ cat opentherm - ttl.md

This adapter is designed to connect an OpenTherm compatible boiler to any TTL port (TTL-USB, ESP8266, ESP32, Arduino, etc).

Same size as popular RS485-TTL module for more compatibility.
